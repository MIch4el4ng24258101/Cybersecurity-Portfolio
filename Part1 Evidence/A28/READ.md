
Password Security Toolkit
Overview
This project is a simple Python-based security solution that helps improve password security. It allows users to:

1. Check password strength
2. Generate a secure password
3. Hash passwords securely with a salt using SHA-256

Why this is a security solution
Weak passwords are a major cybersecurity risk. This tool helps users create stronger passwords and protect stored passwords using hashing.

Features
- Password strength analysis
- Secure random password generation
- Salted SHA-256 hashing

Technologies Used
- Python 3
- hashlib
- secrets
- string
- re
- getpass
